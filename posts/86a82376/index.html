Back2023-07-25

  博客又双叒叕迁移了，不过这次不是简单换了一个系统，而是直接换了“肉身”。原来的LG gram终于因为超期服役带来的体力不支退役了，取而代之的是一个迷你主机，AMD 7940HS，32G+1T的顶配。也算是为我的博客找了一个好人家，没有亏待了这位从2018年开始就陪伴我的老朋友。得益于之前总结的博客迁移技巧，没有花费太多功夫就弄好了，也成功迁移到了新的电脑上，运行丝滑，特此记录。

update 2025.5.19:

秉持着：云同步和所有资料尽可能上云的理念，本地部署的博客显得那么格格不入。了解到 github 的 github action 可以实现自动部署，于是乎，操作了一番成功！见GITHUB ACTION test and guideline，以后再也不需要因为换电脑或者换配置环境而折腾啦！！！
